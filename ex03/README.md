\# ex03 - SOS LED Signal with millis()



\## 实验目的



本实验使用 Arduino 中的 millis() 函数控制 LED 产生 SOS 闪烁信号。



\## SOS 信号规则



SOS 信号由三次短闪、三次长闪、三次短闪组成。每次 SOS 播放结束后，LED 会保持较长时间熄灭，然后重新播放。



\## 时间设置



\- 短闪：200ms

\- 长闪：600ms

\- 闪烁间隔：200ms

\- 一次 SOS 结束后的停顿：1200ms



\## 实验原理



程序使用 millis() 获取系统运行时间，通过比较当前时间和上一次状态切换时间之间的差值，判断是否进入下一个 LED 状态。



相比 delay()，millis() 不会阻塞程序运行，更适合需要稳定计时和多任务扩展的 Arduino 程序。



\## 实验现象



LED 会按照以下顺序循环闪烁：



短、短、短、长、长、长、短、短、短，然后熄灭较长时间。
