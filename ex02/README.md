\# ex02 - LED 1Hz Blink with millis()



\## 实验目的



本实验使用 Arduino 中的 millis() 函数控制 LED 以 1Hz 的频率稳定闪烁。



\## 实验原理



millis() 可以获取程序运行以来经过的毫秒数。通过比较当前时间和上一次状态改变时间的差值，可以判断是否到达 LED 状态切换的时间。



本实验设置间隔为 500ms，使 LED 亮 500ms、灭 500ms，因此完整周期为 1000ms，即 1Hz。



\## 硬件连接



使用 Arduino 板载 LED，连接引脚为 D13。



\## 实验现象



程序上传后，LED 每秒完成一次亮灭变化，表现为稳定闪烁。
