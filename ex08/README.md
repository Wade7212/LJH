\# ex08 - 物联网安防报警器模拟实验



\## 实验目的



使用 ESP32 的触摸引脚、基础 IO 和 Web Server 实现一个简易物联网安防报警器。



\## 实验现象



网页端提供“布防 Arm”和“撤防 Disarm”两个按钮。未布防时触摸引脚无反应；点击布防后，一旦触碰触摸引脚，LED 开始高频闪烁并锁定报警。即使手松开，报警也不会停止，直到网页端点击撤防。



\## 实验原理



程序使用 armed 变量保存布防状态，使用 alarmTriggered 变量保存报警状态。在布防状态下检测触摸引脚，如果触摸值低于阈值，则触发报警。报警触发后，LED 使用 millis() 实现高频闪烁，直到用户点击撤防按钮重置系统状态。
