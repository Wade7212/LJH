\# ex09 - 实时传感器Web仪表盘



\## 实验目的



使用 ESP32 Web Server 实现实时传感器数据上报与网页显示。



\## 实验现象



浏览器打开 ESP32 网页后，页面中央显示一个实时变化的触摸传感器数值。手指逐渐靠近触摸引脚时，数值会变小；手指离开后，数值会恢复。



\## 实验原理



ESP32 在 Web Server 中提供 /touch 接口，实时返回 touchRead() 读取到的触摸传感器数值。网页端使用 JavaScript 的 fetch() 和 setInterval() 周期性请求该接口，并将返回值显示在网页中，实现类似仪器仪表的数据实时监控面板。
